Project Title: Multi-Variable Vector Calculus Library (C++)

Description:
This project is a C++ library that works with 3D vectors used in multivariable calculus. It performs operations like addition, subtraction, dot product, and magnitude. The program uses object-oriented programming concepts such as function overloading and operator overloading. It is simple, reusable, and easy to understand.
